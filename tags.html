Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings including a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data
header header section of page define page heading section
div division of page isolate a block for layout
ol ordered list display ordered list
ul unordered list display unordered list
li list element insert a list element
br line break insert line break
blockquote block quote distinguish content by increasing margins

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong string make string text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
span span isolate a section for styling <span>at least a billion people</span> at least a billion people
li list element display list element <li>element</li>

Document Elements

Tag Name Purpose
html html document type
head headers include information how to interpret document
body document body contents of the page
title pade title title to display
meta date about the content/data include information to help interpret the contents
